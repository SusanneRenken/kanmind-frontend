Personal data is stored only for as long as necessary to provide the functionality of the application. There is no transfer of personal data to third parties.

Your Rights

You have the right to request information about the personal data stored about you, as well as the right to request correction or deletion of this data. Please contact the responsible party using the email address provided above.

Hosting

This application is hosted on servers provided by IONOS SE. When accessing the application, technical access data is processed to ensure the secure and stable operation of the service.

Server Log Files

The hosting provider automatically collects and stores information in server log files, which are transmitted by your browser. These may include:

  • IP address (anonymized where possible)
  • Date and time of the request
  • Requested resource
  • Browser type and version
  • Operating system

This data is processed exclusively to ensure system security and stability and is not combined with other data sources.

SSL / TLS Encryption

For security reasons, this application uses SSL/TLS encryption. This ensures that data transmitted between your browser and the server cannot be read by third parties.

Data Transfer to Third Parties

No personal data is transferred to third parties. Data processing takes place exclusively on the servers of the hosting provider mentioned above.
