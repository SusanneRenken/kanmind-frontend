Privacy Policy

1. General Information

This Privacy Policy explains the nature, scope and purpose of the processing of personal data within the web application KanMind. KanMind is a non-commercial portfolio project with public registration functionality.

3. Hosting

The backend application is hosted on a VPS provided by IONOS SE, Germany. The servers are located in Germany.

When accessing the application, the hosting provider automatically collects server log data including:

  • IP address
  • Date and time of request
  • Requested resource
  • Browser type and version
  • Operating system

Server logs are stored for security and operational purposes. Legal basis: Art. 6(1)(f) GDPR (legitimate interest in secure system operation).

4. Registration and Authentication

Users may register publicly within the application.

The following data is processed:

  • Full name
  • Email address
  • Password (stored in encrypted/hashed form)

Authentication is performed using token-based authentication. Authentication tokens are stored in the user's browser localStorage.

Legal basis: Art. 6(1)(b) GDPR (performance of contract).

5. Application Data

In order to provide project management functionality, the following additional data may be processed:

  • Board titles
  • Task titles and descriptions
  • Due dates
  • Status information
  • Comments
  • Assigned user references

This data is stored in a PostgreSQL database on the same VPS.

6. Email Communication

The application may send system emails (e.g., registration confirmation or password reset). Email services are provided via IONOS SMTP.

Emails may be stored for up to 30 days.

7. Logging

Backend logs may store request-related information including:

  • IP addresses
  • User IDs
  • Error logs

Logs are stored for security and debugging purposes. Legal basis: Art. 6(1)(f) GDPR.

8. Data Retention

Personal data is stored as long as a user account exists. Users may request deletion at any time. No automated deletion routines are implemented.

9. Your Rights

Under the GDPR you have the right to:

  • Access your stored personal data (Art. 15 GDPR)
  • Rectification (Art. 16 GDPR)
  • Erasure (Art. 17 GDPR)
  • Restriction of processing (Art. 18 GDPR)
  • Data portability (Art. 20 GDPR)
  • Object to processing (Art. 21 GDPR)

You also have the right to lodge a complaint with the supervisory authority in Lower Saxony, Germany.

10. SSL / TLS Encryption

The application uses SSL/TLS encryption to protect transmitted data.
